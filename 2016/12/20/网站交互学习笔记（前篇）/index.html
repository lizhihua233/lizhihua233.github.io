二、基础组件

按钮

(1)基础用法

按钮的状态一共有4种，normal、hover、active、disable，如下图：

注（补充说明）：
  1. 正常（normal）：控件处于活动状态，但当前并未使用；
  2. 突出显示（highlighted）：控件正被按住或正被使用；
  3. 禁用（disabled）：按钮未启用且无法使用；
  4. 已选中（selected）：仅特定控件具有改状态，表示控件当前已被选中。
（例如，app里面经常碰到的多选按钮，让你选择你感兴趣的分类，当你click某个或多个后会区别于其他未选中的按钮）

(4)图标式按钮

带图标的按钮可增强辨识度(有文字)或节省空间(无文字)

(5)加载式按钮

点击按钮后进行数据加载操作，在按钮上显示加载状态。

(6)按钮组

表格

用于展示多条结构类似的数据，可对数据进行排序、筛选、对比或其他自定义操作

(1)基础用法

(2)带斑马纹

使用带斑马纹的表格，可以更容易区分出不同行的数据。

(3)带边框

基础的表格展示用法。

(4)带状态

带有状态类，可将表格内容 highlight 显示，方便区分「成功、信息、警告、危险」等内容。


[A] 可定义鼠标 hover 后 highlight 当前行的效果。
[B] 可定义状态类的 highlight 效果，颜色的状态 依次表示:成功的、信息的、警告的、危险 的。

(5)固定行列

表格内容过多时，可选择固定行或列(一般是首行和首列)。


[A] 希望表格在一屏内显示，不需要滚动页面时， 可固定首行，将表格控制在页面可及的范围内展示。
[B] 当表格数据分类过多时，可以固定首列，在内容区域滑动浏览的方式展现。我们更提倡将数据进行合理分类和展现，尽量 避免一屏不能完整展示的情况。
[C] 数据过于复杂时，可将列和行都固定。

(6)单项筛选

用于对某列表格进行单项筛选。


[A] 支持单项筛选的表格，展开筛选项时，默认选中的是「全部」标签，此时该列的名称为默认名称;
[B] 选中某一项后，该列的名称变为该选项的名称 并呈 highlight 状态显示;

(7)多项筛选

用于对某列表格进行多项筛选。


[A] 支持多项筛选的表格，点开后支持 checkbox 多选;
[B] 未选中任意项时「筛选」按钮为 disable 状态，选中任意项后变为 enable 状态;点击后收起 dropdown;
[C]「重置」按钮一直可点击，点击后清空所有筛 选项回到初始状态并收起 dropdown;
[D] 支持多项筛选的表格中，进行筛选后，该列的标题内容不变，但会呈现 highlight 状态;

(8)表格排序

对表格进行排序，可快速查找或对比数据。


[A] 带有排序功能的表格，其排序列表头可以点击 (整块区域可点击)
[B] 从默认状态更改排序后，按照该列进行「正序 排列」，第二次点击后「倒序排列」，若再次点击则回到「正序排列」，以此类推;
[C] 当某列进行排序后，再点击其他列排序，之前 的列回到初始状态;
[D] 排序规则:按照当前列进行排序，当某行的该 列值相同时再按照默认顺序进行排序;
[E] 当某列可以同时排序和筛选时，要区分其点击区域:蓝色部分为「排序」的点击区域，红色部分为「筛选」的点击区域;

(9)修改数据

适用于修改与上下文关联不大的内容，强关联内容建议在 Dialog 或新页面中修改。


[A] 当鼠标移动到可以修改的单元格时，出现「编 辑」按钮;

(10)包含多种数据

表格内可展示多种数据。

(11)交互 - 内容过多的处理

内容过多时的三种处理方式。


[A] 固定列宽，显示所有内容，换行时行高随内容变化而变化;
[B] 固定列宽，多余内容省略，hover 到内容上时用 tooltip 显示所有内容;

日期选择器

用于选择或输入日期

(1.1)选择日

以「日」为基本单位，基础的日期选择控件。


[A] 「今天」有别于其它日期，需要高亮显示
[B] 点击后，即表示选中，关闭dropdown，选择框填充选择的日期
[C] 再次激活后，选中选择框中的「日」。可直接输入，输入规则与「时间选择器 - 通用-手动输入规则」交互类似，年份只限制为「四位数」，月份限制输入「1 - 12」，日期输入限制根据月份做实时判断，例如，在2月份，输入「3」的结果是「03」，不能再输入「0」

(1.2)选择日 - 切换年月

提供两种方式切换年月，可以大跨度地调整时间。


[A] 从日期进入切换年月的视图时，dropdown 的 宽和高保持不变
[B] 点击「年份」进入选择年份的界面，其中交互 与「日期选择器 - 选择年」一致，可参看该部分交互
[C] 已选中的样式
[D] 已选中的样式
[E] 点击「月份」进入选择月份的界面，其中交互 与「日期选择器 - 选择月」一致，可参看该部分交互

(2)选择周

通过扩展日期选择，可以选择固定首尾的周。


[A] 选择周时，整行同时被 hover
[B] 顶部的操作区域与选择日期的交互一致，可切换年月
[C] 以日期范围的形式显示周

(3.1)选择月

以「月」为基本单位，只选择到月，不选择日。


[A] 直接选择月份时，只有文字高亮，类似与日历当中的「今天」
[B] 直接选择年或者月的时候，dropdown 的高度与切换年月不一致

(3.2)选择月 - 切换年

以「月」为基本单位，可以切换年份进行大跨度地选择。


[A] 点击「年份」进入选择年份的界面，其中交互与「日期选择器 - 选择年」一致，可参看该部分交互
[B] 已选中的样式

(4) 选择年

以「年」为基本单位，只选择到年，不选择月、日。


[A] 直接选择年份时，只有文字高亮，类似与日历当中的「今天」
[B] 不可点击，无hover效果

(5)含快捷选项的日期点

左侧区域可配置快捷选项，例如「今天」、「昨天」等。


[A] 顶部的操作区域与「日期选择器 - 选择日-切换年月」的交互一致，可切换年月
[B] 点击后收起 dropdown
[C] 再次激活后，侧栏的快捷选项无选中效果

(6)日期范围 - 往后选择第二个日期

可在一个选择器中便捷地选择一个时间范围。

(7.1)日期范围 - 往后选择第二个日期 - 再次激活

在一个选择器中选择


[A] 激活后，左侧的日历显示起点日期所在的月份，例如:之前选择了「2016/04/24 - 2016/05/24」，则激活后左侧日历显示 「2016年 4月」
[B] 再次点击，就意味着一次新的选择，即重新设置第一个日期

(7.2)日期范围 - 往前选择第二个日期

可以选择任意时间


[A] 往前选择第二个日期，就意味着一次新的选择，即将其设置为第一个日期

(8)含快捷选项的日期范围

可以选择任意时间。


[A] 点击即选中，收起dropdown
[B] 再次激活后，无选中样式
[C] 激活后，左侧的日历显示起点日期所在的月份

表单

由输入框、选择器、单选框、多选框等控件组成，用以收集、校验、提交数据

(1)典型表单

包括各种表单项，如输入框、选择器、开关、单选框、多选框等等。

(2)行内表单

当垂直方向空间受限且表单较简单时，可以在一行内放置表单。

(3)对齐方式

根据具体目标和制约因素，选择最佳的标签对齐方式。


[A] 易视觉扫描，易理解，兼容了不同长度的标签，但是增加了垂直方向的空间，适用于水平方向空间拮据的移动端长表单。
[B] 当表单标签长短差异较大时，采用右对齐，易视觉扫描，但可读性和美观度降低，且当标签长度过长时，浏览表单比较困难
[C] 当表单标签长度基本一致时，建议采用左对齐的方式，此时可读性和美观度较强，同时保证了视觉扫描的快捷
[D] 当表单项目较少时，或者用户对该表单较熟悉时(如登录、注册)，可省略表单标签，节约了空间的同时，提升了浏览效率

(4)表单组

当表单项过多时，为了提升效率，有必要对表单项进行分组。

(5)提示信息

通过提示信息，帮助用户快速准确地完成表单。


[A] 根据提示信息复杂程度选择合适的呈现方式，但要保证整个表单的统一
[B] 轻量级的提示信息: Placeholder，输入内容后，提示信息会被取代
[C] 激活后以 Tooltip 形式呈现的提示信息，仅在激活表单项时出现，提示信息可以比较复杂
[D] 较重的提示信息，默认出现在表单项之下，会一直存在，在表单校验后会被校验信息取代

(6)表单校验

在防止用户犯错的前提下，尽可能让用户更早地发现并纠正错误。


[A] 各种类型的表单项在校验交互上保持一致
[B] 显示提示信息

按钮样式的单选组合。

[B] 鼠标 hover 到「清空」icon 上，按钮样式加深，此时点击可清空选择器内容
[C] 无选中项的选择器，鼠标移上后，倒三角符号不会变化
[D] 点击已选中项的选择器，当鼠标仍在选择器上时，倒三角符号变为「清空」icon
[E] 当鼠标移动到 dropdown 上时，「清空」 icon 变为倒三角符号

(4)基础多选

适用性较广的基础多选，用 Tag 展示已选项。


[A] 点击选择器空白区域可收起/展开 dropdown
[B] 选中后 dropdown 不会自动收起
[C] 激活选择器后，鼠标 hover 到 tag 的删除按钮上，可以点击删除 tag，删除后 dropdown 中对应选项也取消选中状态
[D] 未激活选择器，鼠标 hover 到 tag 的删除按钮时，可以点击删除 tag
[E] 选中项过多时换行显示
[F] 点击 tag 中其他区域，可以展开/收起 dropdown

(5)可搜索多选

可以利用搜索功能快速查找选项。


[A] 激活选择器，hint 内容变为「输入内容搜索」
[B] 选中某一项后，选择器仍处于可搜索状态
[C] 搜索关键词与选项进行模糊匹配，并将匹配词 highlight 显示
[D] 选中搜索结果后，dropdown 还原为未搜索时 的状态
[E] 激活已有选中项的选择器，激活输入框搜索功能，此时无 hint 内容
[F] 淡红色区域为输入区域，当鼠标移动到此区域空白处时，显示为可输入的符号
[G] 淡蓝色区域为点击区域，当鼠标移动到此区域时，显示为手状符号，点击后可收起 dropdown
[H]未做任何选择时收起 dropdown，选择器中只保留之前已选中的 tag，搜索内容将被清空

(6)分组

备选项进行分组展示。

(7)通用-滚动的交互


[A] 待选项大于 7 个时，下拉框呈现最高高度只显示 7.5 个选项
[B] 滚动条默认隐藏，滚动时显示
[C] 当选中项不在下拉框第一屏时，再次激活后将定位到已选中项为最后一项的位置处

(8)通用-键盘操作交互


[A] 点击选择器后，选择器呈现激活状态(或在表单中使用 Tab 键切换选中)
[B] 在已激活的选择器中使用方向键「下」可展开 dropdown 选项
[C] 再使用一次方向键「下」可 hover 顶部第一个选项(此状态仅在没有选中项的选择器中)
[D] 回车选中后，此时选择器仍为选中状态，使用方向键可再次展开下拉框
[E] 在已有选中项的选择器中，使用方向键「下」 后会 hover 已选中项的下一项(如已选中项是最后一项，会 hover 顶部第一项)
注:多选中，会 hover 第一个选项的下一项。
[F] 当 hover 状态在底部最后一项时，再使用方向 键「下」将回到顶部第一项的 hover 状态
[G] 已选中项和 hover 状态重叠的状态

滑块

通过拖动滑块在一个固定区间内进行选择

(1)基础用法

在拖动滑块时，显示当前值。


[A] 初始值默认为0，也可以自定义
[B] hover 和拖动滑块时，显示当前值，滑块变 大。松开后，恢复原尺寸。
[C] disabled 状态下，Cursor 为 not-allowed， 但正常显示当前值。

(2)离散值

选项可以是离散的。


[A] 滑块只能停留在固定的选项上。
[B] 根据实际需求，间断点可选择是否显示。

(3)带输入框

通过输入框设置精准数值。


[A] 滑动或填写输入框时，两者实时同步。配合数字输入框使用。

开关

表示两种相对对立的状态间的切换，多用于触发「开/关」

(1)基础用法


[A] 支持点击和拖动

(2)带文字或图标

用自定义的文字或图标更明确地表示状态。

时间选择器

用于选择或输入时间

(1)固定时间点

提供几个固定的时间点供用户选择。

(2)任意时间点

可以选择任意时间。


[A] 激活选择器后，选择框填充当前的时间，且 「时」被选中，敲击键盘，可直接输入
[B] 激活选择器后，timepicker默认选中当前的时间
[C] 点击「确定」按钮，即表示保存选择框内的内 容且关闭dropdown
[D] 点击「取消」，即清空选择框且关闭 dropdown
[E] 存在禁选项
[F] 滚动时出现滚动条，且可以拖拽，不滚动时， 滚动条隐藏

(3)通用 - 手动输入规则


[A] 当激活输入框时，默认填入此刻时间
[B] 当鼠标 hover 在数列区域时，选中相对应的 「时」「分」「秒」区域，可以手动输入数字 修改时间
[C] 时:能识别 00-23 的任何数组，检测前两位的数组，如果满足则直接填入结果，其余多余输入皆无效。
[D] 分:能识别 00-59 的任何数组，检测前两位的数组，如果满足则直接填入结果，其余多余输入皆无效。
[E] 输入规则同「分」的输入规则

(4)通用 - 限制时间范围

限制时间范围的交互，以只能选择 18:30~20:30 中间的时间举例说明


[A] 当前时间在可选范围内，激活后自动填入当前时间
[B] 在小时刻度上，绝对选不了的时间 disable 状态，分钟和秒钟刻度不做此限制
[C] 当前时间在可选范围前，激活后填入最小可选时间
[D] 当前时间在可选范围后，激活后填入最大可选时间
[E] 改动分钟后，若此时时间小于可选范围，则将改动 的分钟自动调整为可选的最小分钟(秒钟同理)
[F] 改动分钟后，若此时时间大于可选范围，则将改动的分钟自动调整为可选的最大分钟(秒钟同理)

(5)固定时间范围 - 先选择开始时间

先选择开始时间，结束时间内备选项的状态会随之改变。


[A] 选择开始时间后，自动激活结束时间选择器
[B] 在开始时间之前的项禁选

(6)固定时间范围 - 先选择结束时间

开始时间的备选项不随结束时间的选择改变


[A] 开始时间不会因结束时间产生变化，随时都可以自由选择。选择后，结束时间会根据开始时间进行判断，如果在开始时间后则成立;如果在开始时间前则清空

(7)任意时间范围

可选择任意的时间范围。


[A] 要保证结束时间在开始时间之后，具体过程参看「时间选择器 - 通用-限制时间范围」交互

上传

通过点击或者拖拽上传文件

(1)点击上传多个文件

适用广泛的基础用法。


[A] 说明上传文件的格式、尺寸、大小等要求，文字尽可能精简
[B] 触发上传操作的按钮根据实际情况可能不同，也可以使用文字按钮和 Ghost 按钮，按钮内容可以自定义
[C] 上传成功，恢复到常规状态，通过 Alert 予以提醒。若因为服务器等原因上传失败，需要通过 Alert 予以提醒，若因为上传文件不符合要求，则予以报错。
[D] hover 文件名称时，对应的操作出现(需要动效配合)，且紧跟在文件名称之后
[E] 如再次上传同一文件名的文件，则在原文件位置重新上传一次
[F] 点击「删除」，进行二次确认，确定即恢复到最初的默认状态

(2)点击上传单个文件

点击 button 进行文件上传，上传成功后，button 隐藏。


[A] 对于上传单个文件，选择好文件后，文件代替 「点击上传」按钮，显示进度
[B] hover 文件名称时，对应的操作出现(需要动效配合)，且紧跟在文件名称之后。点击「重新上传」，即重新触发一次上传操作。

(3)拖拽上传

可将文件拖入指定区域进行上传。


[A] 如果对上传文件有要求，比如类型/大小等，可以在这里体现。如果没有要求，可以省略。若 对尺寸有要求，那么该上传区域长宽比要与要求的长宽比一致
[B] 当文件拖动至浏览器区域后，该区域即出现对应变化，提示文件拖动的最终区域
[C] 将文件进一步拖动至最终区域后，有对应的视觉效果，释放后即开始上传文件
[D] 上传结果显示在上传区域下面

(4)上传单个图片

针对图片类型文件的上传，上传后显示缩略图。


[A] 「点击或拖拽上传」交互参看「上传 - 拖拽上传」，同时支持点击上传
[B] hover该区域后，出现对应操作和文件名称
[C] hover 后 tooltip 显示按钮名称，分别是「重新上传」、「查看图片」、「删除图片」
[D] 当名称过长时，hover 后以 tooltip 显示全称

(5)上传多个图片

可以上传多个图片，上传后显示缩略图。


[A] 拖拽上传的小型版本，只有图标，没有说明文字
[B] 「点击或拖拽上传」交互参看「上传 - 拖拽上传」
[C]最新的上传结果排列在最前面
[D] hover 图片后有对应的效果，具体参看 「上传 - 点击或拖拽上传单个图片」交互
[E] 再次拖拽，即在原区域上添加白色遮罩以显示内容，其它参看「上传 - 点击或拖拽上传多个文件」交互
